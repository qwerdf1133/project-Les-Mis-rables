기존 JAVA 
JVM의 고질적인 문제는 시동시간 VM을 반복해서 종료, 
그리고 요청이 왔을 때 구동
 -> 대략 3초의 시간이 필요했다.

GraalVM의 VM은 JVM 내부에서 실행된다는 사실에서 비롯 되었으며 
GraalVM은 Oracle이 만든 JVM과 JDK으로 
애플리케이션 성능과 효율성의 향상을 제공하는 고성능 런타임이다.
